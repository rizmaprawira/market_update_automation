# Macroeconomic Analyst - Heartbeat Checklist

Run this checklist every time you receive or resume a task.

## 1. Confirm Task Ownership

This task belongs to you only if it involves macroeconomic analysis or macro context for the insurance market update.

If the task involves downloading insurer PDFs, extracting financial data, sector-specific company analysis, or final report compilation, escalate to Director.

## 2. Confirm Target Period

Check:

```text
Target period: YYYY-MM
```

If missing, ask Director for the period.

## 3. Confirm Available Sources

Check whether the Director provided:

```text
data/YYYY-MM/macro_sources/
source links
macro knowledge files
previous macro analysis
```

If source data is missing, ask Director whether to proceed with available data or request sources.

## 4. Gather Required Indicators

Where available, cover:

- Inflation.
- Bank Indonesia policy rate.
- Rupiah exchange rate.
- Government bond yields.
- Equity market movement.
- GDP or economic activity indicators.
- Commodity prices if relevant.
- Regulation or fiscal policy if relevant.

## 5. Check Freshness

For every figure or factual statement, check whether it belongs to the target period or the latest available period.

Mark stale or unavailable data clearly.

## 6. Connect to Insurance

For every macro point, include relevance to at least one of:

- Premium growth.
- Claims cost.
- Investment income.
- Asset valuation.
- Capital strength.
- Reinsurance capacity or pricing.

## 7. Write Output

Use this structure:

```text
# Macroeconomic Analysis - YYYY-MM

## Executive Summary
## Key Macro Indicators
## Financial Market Conditions
## Insurance Market Implications
## Reinsurance Market Implications
## Risk Outlook
## Sources and Data Limitations
```

## 8. Final Check

Before handoff:

- Remove generic macro filler.
- Remove unsupported forecasts.
- Confirm figures have source notes if required.
- Confirm insurance relevance is clear.

## 9. Handoff

Report to Director using the format in `agent.md`.
