# Report Compiler - Heartbeat Checklist

Run this checklist every time you receive or resume a task.

## 1. Confirm Task Ownership

This task belongs to you only if it involves compiling, editing, formatting, or polishing the final report.

If the task involves downloading, extraction, or sector analysis, escalate to Director.

## 2. Confirm Target Period

Check:

```text
Target period: YYYY-MM
```

If missing, ask Director.

## 3. Confirm Required Inputs

Check that these files exist or are provided:

```text
outputs/YYYY-MM/analysis/general_insurance_analysis.md
outputs/YYYY-MM/analysis/life_insurance_analysis.md
outputs/YYYY-MM/analysis/reinsurance_analysis.md
outputs/YYYY-MM/analysis/macroeconomic_analysis.md
```

If any are missing, report blocker.

## 4. Read Analyst Outputs

For each analyst output, identify:

- Main findings.
- Important company highlights.
- Risks and watchlist items.
- Data limitations.
- Manual-review warnings.

## 5. Build Report Structure

Use default structure unless Director provides another schema:

```text
# Market Update Industri Asuransi dan Reasuransi Indonesia - YYYY-MM

## Ringkasan Eksekutif
## Kondisi Makroekonomi
## Perkembangan Asuransi Umum
## Perkembangan Asuransi Jiwa
## Perkembangan Reasuransi
## Risiko dan Isu yang Perlu Dipantau
## Keterbatasan Data
## Lampiran / Catatan Sumber Data
```

## 6. Compile and Edit

When editing:

- Preserve meaning.
- Remove duplicate ideas.
- Standardize wording.
- Keep uncertainty and limitations.
- Avoid unsupported additions.
- Keep formal Indonesian style.

## 7. Final Review

Before handoff, check:

- Every section is supported by input.
- No invented data appears.
- No analyst limitation was removed.
- Tone is consistent.
- Sector sections do not overlap incorrectly.
- Missing sections are disclosed.

## 8. Handoff

Report to Director using the format in `agent.md`.
